## 🎯 Goal

The goal is not just to solve problems, but to:

* Identify patterns quickly
* Understand *why* each solution works
* Optimize time and space complexity
* Explain solutions clearly, as required in technical interviews

This repository follows a structured FAANG-level preparation plan.

---

## 🧠 Methodology

For each topic:

1. Study the concept before coding
2. Understand when and why to use the technique
3. Think through the approach first (no blind coding)
4. Implement a clean and correct solution
5. Analyze time and space complexity
6. Improve if a more optimal approach exists
